Coupling MetaSWAP and MODFLOW
-----------------------------

Coupling MetaSWAP and MODFLOW is done using ``primod`` package, which needs to
be installed separately. Documentation for this package can be found at the
`iMOD Documentation
<https://deltares.github.io/iMOD-Documentation/primod_api/>`_. An example on how
to use this package can be found in `this example
<https://deltares.github.io/iMOD-Documentation/coupler_metamod_example.html>`_
