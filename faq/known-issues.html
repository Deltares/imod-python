Known Issues#

MODFLOW 6 versions#

iMOD Python is tested to work with MODFLOW 6 versions:

  • All 6.2, 6.3 versions

  • 6.4.0, 6.4.2

  • All 6.4, and 6.5 versions

Confirmed to not work:

  • 6.4.1

iMOD5 Backwards compatibility#

For a detailed description of known issues with iMOD5 backwards compatibility, see iMOD5 backwards compatibility.
